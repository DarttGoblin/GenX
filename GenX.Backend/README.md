🎬 GenX: AI-Powered Poster Generation
GenX is a deep learning project designed to automatically generate visually compelling posters for movies, series, or shows based on a given title and storyline. By fine-tuning a pretrained diffusion model, GenX produces posters that reflect the genre, tone, and narrative elements of the input text.

🚀 Project Overview
GenX uses text-to-image generation to produce AI-generated posters aligned with a story's content. It learns from a dataset of real-world posters paired with their titles and descriptions, capturing patterns in:

Artistic composition

Genre-specific aesthetics

Visual storytelling

Typography balance

The final model is capable of producing diverse poster styles with strong semantic alignment to the input prompts.

🧠 Core Features

🎨 Text-to-Image Generation: Input a title and synopsis, get a high-quality AI-generated poster.

🖼️ Fine-tuned Diffusion Model: Uses pretrained models from Hugging Face Diffusers for image generation.

🧾 Poster Semantics: Understands narrative themes, mood, and characters from text.

🎭 Genre Adaptability: Adapts visual style according to genre (thriller, drama, sci-fi, etc).

🛠️ Future-Ready: Designed for extensions like style transfer, interactive refinement, or theme-specific poster batches.

🧪 Lab Implementation
Utilizes Stable Diffusion.

Text conditioning via pretrained language encoders.

Fine-tuned on a curated dataset of posters, titles, and plot summaries.

Training involves optimizing cross-modal alignment between text and visual outputs.
